# Wrap Up Data

Post-weigh-in wrap-up JSONs generated by the `/wrap-up` skill.

- One file per event: `<event-slug>.json`
- Schema: `src/types/wrap-up.ts` → `WrapUpData`
- Rendered by: `src/app/[locale]/hub/analise/evento/[eventSlug]/wrap-up/page.tsx`

To generate a new wrap-up, run the slash command:

```
/wrap-up <event-slug>
```

Example: `/wrap-up prochazka-vs-ulberg`
